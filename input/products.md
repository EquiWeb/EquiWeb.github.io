## EquiForge
A x402 SaaS for offering cloud hosting services to agents, facilitating [x402](https://www.x402.org/x402-whitepaper.pdf) for receiving payments.  
[GitHub](https://github.com/EquiWeb/EquiForge)  
[equi-forge.vercel.app](https://equi-forge.vercel.app)

## EquiCode
Experimental coding agent exploring advanced context management strategies for agentic performance, especially in long, complex sessions and large codebases.  
[GitHub](https://github.com/EquiWeb/EquiCode)

## EquiDrive
Android-based ADAS system, using a multi-headed vision model for trajectory estimation and segmentation. Targets low latency model performance with high accuracy.  
*Coming soon*

## Mimir AIP
Data aggregation, processing, and analysis toolkit with ontology-backed reasoning.  
[Mimir-AIP Python — GitHub](https://github.com/Mimir-AIP/Mimir-AIP) — *Archived*  
[Mimir-AIP Go — GitHub](https://github.com/Mimir-AIP/Mimir-AIP-Go)

## EquiStop
Experiments with detecting and reversing AI-slop content, builds on [Klyro](https://github.com/CiaranMcAleer/Klyro). Current focus is on content classification, with the next step being a small transformer model to de-slop content and integrate back into Klyro.  
[GitHub](https://github.com/EquiWeb/EquiStop)

## EvoMind
Family of LLMs focused on improving reasoning performance via self-distillation. Targeting GPQA Diamond, AIME 2025, SWE-Bench, HLE, and U-MATH.  
*Coming soon*

## EquiBench
LLM benchmark designed to test performance in code understanding, planning, and coding. An agent is given a PR from a large codebase, asked to generate an implementation plan; a second agent implements from that plan, and output is compared against the original.  
[GitHub](https://github.com/EquiWeb/EquiBench)

## EquiDesk
Framework for building AI personal assistants and co-workers with persistent memory, autonomous task execution, and MCP tool integration. Explored continuous internal performance benchmarking and agent self-improvement.  
*Archived — codebase release and write-up coming soon*

## EquiAlt
Ongoing project to build a ML dataset for alt-text repair.  
*Coming soon*

## EquiChat
AI chat app built in React Native (Expo), focused on accessibility features.  
*Archived — codebase release coming soon*

## EquiScout
Distributed UAV sensing and tracking system leveraging mesh communication (inspired by [Reticulum](https://reticulum.network)) and edge nodes with multi-modal COTS sensors.  
*Coming soon*
