# EquiWeb

## EquiForge
A x402 SaaS for offering cloud hosting services to agents, facilitating [x402](https://www.x402.org/x402-whitepaper.pdf) for recieving payments. 
[EquiForge](https://equi-forge.vercel.app)
[Github](https://github.com/EquiWeb/EquiForge)

## EquiCode
Experimental coding agent exploring advanced context management strategies for agentic performance, especially in long, complex sessions and large codebases.
[Github](https://github.com/EquiWeb/EquiCode)

## EquiDrive
Android-based ADAS system, using a multi-headed vision model for trajectory estimation and segmentation. Targets low latency model performance with high accuracy.

- [Mimir-AIP Go](https://github.com/Mimir-AIP/Mimir-AIP-Go): Go rewrite for scalable, ontology-backed analysis

## EquiStop
Experiments with detecting and reversing ai-slop content, builds on [Klyro](https://github.com/CiaranMcAleer/Klyro).
- Current focus is on the classification of content
- Once AI slop content can be accurately classified, next step will be training a small transformer model to 'de-slop' 
- Finally integrate into Klyro(replacing/augmenting existing summary appraoch) to detect slop content as you browse, replace with a 'de-slop' version(inference via HF transformers.js)
[Github](https://github.com/EquiWeb/EquiStop)

## EvoMind
Family of LLM's(large language models) focused on improving reasoning performance by utilising self-distillation
- Goals: Improve performance at reasoning heavy benchmarks including STEM heavy bencharks such as(GPQA Diamond, AIME 2025, SWE Bench, HLE, U-MATH)
__Coming Soon__

## EquiBench
LLM bencchmark designed to test LLM performance in both code-understanding and planning, as well as coding.
- LLM agent given access to a PR of a large codebase, asked to generate implementation plan
- PR reverted and LLM agent given access to plan generated by previous agent, output compared against original PR implementation
[Github](https://github.com/EquiWeb/EquiBench)

## EquiDesk
AI employee/personal assistant/co-worker. A framework for building AI personal assistants and co-workers. EquiDesk provides LLM-backed agents with persistent memory, autonomous task execution, and seamless tool integration through the Model Context Protocol (MCP).
- Explored concept of continuous internal performance benchamrking and agent self-improvement system(inspired by paper: [Live-SWE-agent](https://arxiv.org/abs/2511.13646))
__Archived: Codebase release and internal benchmark performance post coming soon__
